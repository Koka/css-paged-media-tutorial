A tutorial and showcase for CSS Paged Media
===========================================

#.. note::
#
#  There will be a `CSS Paged Media hands-on training
#  <http://www.xmlprague.cz/day3-2016/#css>`_ at XML Prague 2016 on Saturday,
#  13.02.2016.  The foundation for this training is the content and examples on
#  the website. You can download the complete material from `Github
#  <https://github.com/zopyx/css-paged-media-tutorial/releases>`_.  If you plan
#  to actively participate in the training, please install one of the converters
#  (Antennahouse, PrinceXML or PDFreactor) first and download the material
#  before the conference in order to safe time and bandwidth. Participants
#  should have basic HTML/XML and CSS knowledge. The first half of the training 
#  will cover the basics of CSS Paged Media, usage of tools and a walk-through
#  the basic features. The second half will be more open to discussion, individual
#  questions etc. Changes on short notice are possible depending on number of 
#  participants etc...this will be a very open and flexible event.
